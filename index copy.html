ESP32 Web Flasher

Instructions

  1. Connect your ESP32 device via USB
  2. Click Connect button
  3. Select firmware version
  4. Click Flash Firmware button
  5. Wait for the process to complete
  6. Click Disconnect when done

Note: Browser will ask for serial port permission when connecting. Make sure your ESP32 is in bootloader mode if flashing fails.
